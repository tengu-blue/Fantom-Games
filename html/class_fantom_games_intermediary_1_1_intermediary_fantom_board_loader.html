Detailed Description

A loading class for the Fantom playing board using a special text file format. Requires the path to the text file. Any lines begging with a double slash are considered comments and are skipped. Expects the first line to contain an integer number for the number of tiles to be loaded
The following lines for each give information about each of the tiles.
